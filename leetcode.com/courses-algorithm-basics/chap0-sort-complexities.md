## Sort Complexities
 
[Sort Complexities @ leetcode.com](https://leetcode.com/articles/sort-complexities/)

## Question

```
Sort the following complexities:

- e^n
- n !
- n^100
- n^n
​​```


## Analysis

![img](https://discuss.leetcode.com/uploads/files/1461676022218-big-o-complexity.png)

## Answer

```
n^100 < e^n < n! < n^n
```
